Selenophile 0.1.1

Selenophile 0.1.1 improves the first public beta with automatic update checks, clearer localized status text, and better settings feedback for Moonraker connection issues.

Highlights

  • Enable Sparkle automatic update checks with the published GitHub Pages appcast.
  • Keep printer status labels, widget snapshots, and menu bar summaries aligned with the selected app language.
  • Present Moonraker connection errors with localized, user-facing messages while keeping runtime logs in English.
  • Resolve conflicting settings feedback when save and test-connection actions happen close together.
  • Use English release notes for Sparkle update presentation.

Current Limitations

  • This beta still focuses on monitoring. It does not upload G-code, start prints, pause, resume, cancel prints, or modify Klipper configuration.
  • The local release build is ad-hoc signed and is not notarized with a Developer ID certificate.
