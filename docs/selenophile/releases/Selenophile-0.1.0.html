Selenophile 0.1.0

Selenophile 0.1.0 is the first public beta for Apple Silicon Mac users running Moonraker / Klipper. It is ready to use as a daily menu bar monitoring tool, while the app remains in beta and will continue to improve distribution, update handling, and detail-level polish.

Highlights

  • View live Klipper printer status from the macOS menu bar.
  • Show print progress, file name, elapsed time, remaining time, layer count, print speed, nozzle temperature, and bed temperature.
  • Fetch camera snapshots through Moonraker for a quick view of the current print.
  • Configure connection details, status refresh policy, launch at login, interface language, appearance mode, and color palette.
  • Use the built-in debug log window to inspect Moonraker connection, retry, status update, and camera request issues.
  • Integrate Sparkle 2 update support for future automatic update distribution.

Current Limitations

  • This version focuses on monitoring. It does not upload G-code, start prints, pause, resume, cancel prints, or modify Klipper configuration.
  • Only Apple Silicon Macs running macOS 14 or later are supported.
