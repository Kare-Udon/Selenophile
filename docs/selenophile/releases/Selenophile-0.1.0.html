Selenophile 0.1.0

Selenophile 0.1.0 是第一个公开 beta 版本，面向使用 Moonraker / Klipper 的 Apple Silicon Mac 用户。这个版本已经可以作为日常菜单栏监控工具使用，但仍处于 beta 阶段，后续会继续完善发布、更新和细节体验。

主要功能

  • 在 macOS 菜单栏中查看 Klipper 打印机的实时状态。
  • 显示打印进度、文件名、已用时间、剩余时间、层数、打印速度、喷嘴温度和热床温度。
  • 通过 Moonraker 获取相机快照，便于快速确认当前打印画面。
  • 提供连接设置、状态刷新频率、登录启动、界面语言、外观模式和配色方案。
  • 内置调试日志窗口，帮助排查 Moonraker 连接、重试、状态更新和相机请求问题。
  • 接入 Sparkle 2 更新框架，为后续自动更新分发做好准备。

当前限制

  • 当前版本专注于监控，不提供上传 G-code、开始打印、暂停、恢复、取消打印或修改 Klipper 配置等控制功能。
  • 仅支持 Apple Silicon Mac 和 macOS 14 或更高版本。
